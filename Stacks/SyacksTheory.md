Stack

- It's a data structure also called as abstract data type.
- Follows LIFO (last in first out).
- Methods associated (Time Complexity should be O(1))
  - push(E e) [insert the element in stack]
  - pop() [remove last inserted element in stack]
  - peek() [access last inserted element in stack]
- Implementation is done majorly by Linked List or Array depending on languages.
